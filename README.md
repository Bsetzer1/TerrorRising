# TerrorRising
Game project for 3D game design class.
This game is a round based zombie survival game.
See how long you can surive? or escape?

Controls:
WASD - Movement
Mouse - Camera Movement
Left Click - Fire
R - Reload
E - Interact
Space - Jump

Cheats:
T - GodMode (MAX HEALTH)
Y - Unlimited Ammo
U - Max Points
I - Max Damage (One Shot)
