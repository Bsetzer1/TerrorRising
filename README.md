# TerrorRising
Game project for 3D game design class.
This game is a round based zombie survival game.
See how long you can survive? or escape?

Controls:
WASD - Movement
Space - Jump
LShift - Sprint
Mouse - Camera Movement
Left Click - Fire
R - Reload
E - Interact

Cheats:
T - GodMode (MAX HEALTH)
Y - Unlimited Ammo
U - Max Points
I - Max Damage (One Shot)
